# 回到顶部

# 思路

  1. 基本功能：点击可以回到顶部
  2. 只有当滚动到一定高度，才出现回到顶部组件
  3. 支持自定义回到顶部的插槽，用于自定义结构和样式
  4. 提供触发回到顶部事件之前和完成的钩子
  5. 防抖节流处理，避免多次点击

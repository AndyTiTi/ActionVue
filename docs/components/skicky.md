## API

| 参数        | 说明   | 类型       | 默认值 |
| ----------- | ------ | ---------- | ------ |
| position    | 组件的吸附方向，顶部或者底部(top/bottom) | String | top |
| distance    | 距离 | Number | 0          |
| customClass | 吸附时自定义类名，方便指定样式 | String | stuck      |

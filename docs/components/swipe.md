## API

| 属性   | 说明         | 类型    | 默认值 |
| :----- | :----------- | :------ | :----- |
| `loop` | 开启无限轮播 | Boolean | `true` |
| `autoPlay` | 是否自动切换 | Boolean | `false` |
| `interval` | 是否自动切换的时间间隔，单位 ms | Number | `3000` |


### 事件

| 事件名称   | 说明         | 回调参数    |
| :----- | :----------- | :------ | 
| `change` | 幻灯片切换时触发 | 目前激活的幻灯片的索引，原幻灯片的索引 |

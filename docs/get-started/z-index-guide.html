# z-index 层级规范

actionvue 使用了较为保守的层级设置以确保用户可以轻松覆盖原样式。

各组件 z-index 数值如下：

Component z-index
Sticky 100
BackTop 100
Toast 100
Message 100
Notice 100
Popover 100
Github Corner 20

# 快速上手

基于 Vue.js 开发的一套组件库. :stuck_out_tongue_winking_eye:

# 安装

使用 npm 安装

npm i actionvue --save

或使用 yarn 安装

yarn add actionvue

# 导入组件

# 完整引入

在 main.js 中写入以下内容，就可以将 actionvue 完整引入使用：

import Vue from 'vue';
import actionvue from 'actionvue';

Vue.use(actionvue);
